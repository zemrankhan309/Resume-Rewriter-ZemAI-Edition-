# 🚀 ZemAI: AI Resume Rewriter & ATS Score

**ZemAI Edition** is a lightweight Chrome extension that uses **Google Gemini Flash** to rewrite your resume to match any job description and generate an instant ATS optimization report—100% locally in your browser.

## ✨ Key Features

* **AI Rewriter:** Instantly tailors your resume for any job description.
* **ATS Diagnostics:** Get your Match %, Keyword Density, Missing Skills, and Red Flags.
* **Easy Export:** One-click Copy or Download directly as a Word document (`.docx`).
* **Zero Config Boilerplate:** Just paste your free Gemini API key and go.
